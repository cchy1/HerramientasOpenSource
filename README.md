# 🧰 HerramientasOpenSource

Este repositorio reúne una colección de herramientas de software libre y configuraciones reutilizables para el desarrollo de proyectos de automatización, domótica rural, monitoreo ambiental, inteligencia artificial aplicada y gestión técnica desde macOS.

Está pensado como una **caja modular** que puede alimentar distintos proyectos prácticos, como sistemas de seguridad, cercos virtuales ganaderos, asistentes clínicos y automatización en Home Assistant.

---

## 🗂️ Estructura del repositorio

### 🏡 `home_assistant/`
Automatizaciones, blueprints y configuraciones para domótica. Incluye control de luces, sensores, cámaras, y notificaciones desde Home Assistant.

### 🌐 `esp32_lora/`
Configuraciones para nodos ESP32 con LoRa. Incluye archivos ESPHome, proyectos PlatformIO y documentación para sensores rurales, collares ganaderos y nodos autónomos.

### 🧠 `scripts_python/`
Scripts en Python para tareas comunes como:
- Transcripción de audio (Deepgram, Whisper)
- Análisis de sentimiento (modelos tipo BETO)
- Automatización clínica o análisis textual

### 🔁 `n8n_flows/`
Flujos de automatización para n8n. Incluye integración con Gmail, Google Drive, Google Tasks, Webhooks y APIs externas.

### 🍏 `utilidades_mac/`
Scripts y configuraciones útiles para trabajar en macOS con chip M1/M2. Incluye instalación de herramientas, configuración de terminal y scripts de mantenimiento.

---

## 💡 Propósito

- Aprovechar software open source de forma práctica
- Reutilizar código y configuraciones entre proyectos
- Crear una base técnica portátil y adaptable
